CheapoDC WebFlash

WebFlash Installer

This WebFlash installer is for the Cheapo Dew Controller (CheapoDC). For the latest information about CheapoDC, as well as how to configure the device, see the CheapoDC GitHub repository.

CheapoDC was developed for the ESP32-C3 SuperMini and this installer will only work for ESP32-C3 based modules. WebFlash will erase all flash memory on your device then do a full install to both the firmware and data partitions. If upgrading a device, make sure you backup your CDCWiFi.json and CDCConfig.json configuration files before flashing your CheapoDC.

Connect your ESP32 to your computer via USB to use WebFlash. Make sure that no other programs are connected to the same port for your device. Once the WebFlash is complete goto First Time Device Configuration to setup your CheapoDC.

Click Connect to start WebFlash
